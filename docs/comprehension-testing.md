# Guided-story comprehension test

Use this protocol with 5–8 participants before changing the chapter structure again.

## Tasks

1. Explain what changed after moving the first chapter’s primary control.
2. Find the site’s current explanation for why matter survived annihilation.
3. Skip directly to the abiogenesis question.
4. Return to the previous chapter using browser navigation.
5. Identify whether one prominent claim is an observation, a model, or an open question.

## Observation rubric

Record success without assistance, hesitation points, incorrect interpretations, and the participant’s final explanation. Do not lead participants toward interface labels. After each task, ask: “What do you think the site is claiming, and how certain is that claim?”

## Privacy-conscious events

The application may emit chapter views, primary-experiment engagement, deep-lab toggles, and chapter completion. Do not record raw parameter values, free-text answers, fingerprints, or personal identifiers by default.
